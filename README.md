# Tic-Tac-Toe Full-Stack

This repository contains a starter full-stack Tic-Tac-Toe app:

- `backend` — Express + Socket.IO server, Prisma schema + seed, tests.
- `frontend` — Vite + React + TypeScript client.

See `backend/README.md` and `frontend/README.md` for local dev steps.

Important: provide your `DATABASE_URL` (Supabase/Postgres) in `backend/.env` before running migrations.
